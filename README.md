## Intro
This download cleaning project cleans out your downloads folder<br>
NOTE: YOU MUST CHANGE THE PATH TO THE DOWNLOADS DIRECTORY IN THE SORTER.PY
It creates a couple of folders in the downloads folder
* Audios
* Documents
* Folders
* Images
* Others
* Videos
* Zips

After doing that, it catagorizes the items accordingly

## Usage
You must clone this repository to your computer, and then just run<br>
```bash
cd download-cleaner
python3 sorter.py
```

You get asked your email and password, then it starts to organize the Downloads into each folder
<br>

### Installation
#### Linux
To install python, type the following into your terminal
```bash
sudo apt-get install python3
```

If you get any errors saying that you don't have the module decouple, then you must run
```bash
pi3 install decouple
```
